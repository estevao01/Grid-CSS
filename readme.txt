Explicando as classes:

.section: Cria uma seção com altura minima de 200px e largura total para receber os demais elementos.

.line: Cria uma linha para trabalhar com as columns-1 - 12, com margin nas laterais

.line-fw:  Cria uma linha para trabalhar com as columns-1 - 12, sem margin nas laterais

.col-1 ao .col-12: Classes responsável por controlar o Grid, sendo col-1 equivalente a 1/12 da tela do dispositivo(baseando na largura da line usada) e col-12 equivalente a largura total do line

.center-v: Centraliza o elemento verticalmente

.center-h: Centraliza o elemento horizontalmente

.center-vh: Centraliza o elemento tanto na vertical quanto na horizontal

.full-h faz com que o elemento oculpe 100% da altura da tela.
